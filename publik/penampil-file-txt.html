Cara Penggunaan

1. Pilih File TXT: Klik tombol "Pilih file TXT" dan pilih file dengan ekstensi `.txt` dari perangkat Anda.

2. Tampilkan Konten: Setelah file dipilih, klik tombol "Tampilkan Teks".

3. Lihat Hasil: Konten dari file TXT Anda akan ditampilkan di area tengah, dengan setiap baris yang mengandung ':' akan diformat sebagai judul dan nilai.

4. Hapus Konten: Gunakan tombol "Hapus Konten & File" untuk membersihkan tampilan dan mereset pilihan file.

Peringatan: Aplikasi ini hanya mendukung file teks biasa (.txt). File dengan format lain tidak akan dapat diproses.

Tentang Teknologi Ini

Aplikasi Penampil File TXT ini dibangun menggunakan teknologi web dasar: HTML untuk struktur konten, CSS untuk gaya dan tata letak, serta JavaScript untuk fungsionalitas membaca dan menampilkan file.

Tujuan utamanya adalah untuk menyediakan alat sederhana dan cepat untuk melihat isi file teks secara langsung di browser, tanpa perlu mengunduh atau menggunakan aplikasi desktop terpisah. Ini sangat berguna untuk melihat log, konfigurasi, chat wa yang diekspor, atau data teks sederhana lainnya.
